## 使用提示

1. **開始新對話時**：貼上此模板，Claude 會自動理解案件背景
2. **對話結束前**：請 Claude「請更新案件記憶模板的第七節對話記錄摘要」
3. **定期維護**：每週檢視一次，確保資訊正確
4. **多案件管理**：每個案件建立一個獨立的 Claude Project
